Le système GUI

Permet de récupérer les différents GUI ainsi que d'effectuer certaines actions sur les différents éléments graphiques.


Méthodes​

Voici la liste des méthodes dont vous aurez besoin pour vos scripts

GetGUI(name: string)​

Paramètres:

  • name @string: Le nom du système que l'on veut récupérer

Cette fonction récupère un ScreenGUI de l'interface utilisateur en fonction de son nom.

Exemple:

local GUI = self.FGS:GetSystem("GUI")

local LeGUIqueJeVeux = GUI:GetGUI("NomDeMonScreenGUI")

Tween(element: Instance, data: table, t: number, easingStyle: Enum, easingDirection: Enum)​

Paramètres:

  • element @Instance: L'élément que l'on veut tween
  • data @table: La table contenant les données de notre tween
  • t @number: Le temps que notre tween devrait prendre
  • easingStyle @Enum (Optionnel)
  • easingDirection @Enum (Optionnel)

Cette fonction permet de tween un élément donné.

Exemple:

local MonTexte = LeGUIqueJeVeux:FindFirstChild("Text")

-- Tween le texte
local tween = GUI:Tween(
    MonTexte,
    {
        Color3 = Color3.fromRGB(255, 134, 32)
    },
    .15
)

tween:Play()

ChangeFrame(previous: Frame, target: Frame)​

Paramètres:

  • previous @Frame: Le frame actuel à cacher
  • target @Frame: Le frame cible à afficher

Cette fonction permet de changer l'affichage d'un frame à un autre.

Exemple:

local previousFrame = LeGUIqueJeVeux:FindFirstChild("PreviousFrame")
local targetFrame = LeGUIqueJeVeux:FindFirstChild("TargetFrame")

GUI:ChangeFrame(previousFrame, targetFrame)

CloseWindow(window: CanvasGroup)​

Paramètres:

  • window @CanvasGroup: La fenêtre à fermer

Cette fonction permet de fermer une fenêtre GUI qui a été enregistrée par :RegisterWindow.

Exemple:

local window = LeGUIqueJeVeux:FindFirstChild("MyWindow")

GUI:CloseWindow(window)

OpenWindow(window: CanvasGroup)​

Paramètres:

  • window @CanvasGroup: La fenêtre à ouvrir

Cette fonction permet d'ouvrir une fenêtre GUI qui a été enregistrée par :RegisterWindow.

Exemple:

local window = LeGUIqueJeVeux:FindFirstChild("MyWindow")

GUI:OpenWindow(window)

ToggleWindow(window: CanvasGroup)​

Paramètres:

  • window @CanvasGroup: La fenêtre à basculer

Cette fonction permet de basculer la visibilité d'une fenêtre GUI qui a été enregistrée par :RegisterWindow.

Exemple:

local window = LeGUIqueJeVeux:FindFirstChild("MyWindow")

GUI:ToggleWindow(window)

RegisterWindow(window: CanvasGroup)​

Paramètres:

  • window @CanvasGroup: La fenêtre à enregistrer

Cette fonction permet d'enregistrer une fenêtre GUI.

Exemple:

local window = LeGUIqueJeVeux:FindFirstChild("MyWindow")

GUI:RegisterWindow(window)

RegisterButton(button: TextButton, hoverEffect: string, callbackClick: function)​

Paramètres:

  • button @TextButton: Le bouton à enregistrer
  • hoverEffect @string: L'effet de survol à appliquer ("Zoom" ou "Opacity")
  • callbackClick @function: La fonction à appeler lors du clic sur le bouton

Cette fonction permet d'enregistrer un bouton avec un effet de survol et un rappel de clic.

Exemple:

local button = LeGUIqueJeVeux:FindFirstChild("MyButton")

GUI:RegisterButton(button, "Zoom", function()
    print("Bouton cliqué")
end)

BindHoverEffect(button: TextButton, content: Instance, scale: UIScale, hoverEffect: string)​

Paramètres:

  • button @TextButton: Le bouton auquel lier les effets de survol
  • content @Instance: L'instance de contenu à l'intérieur du bouton
  • scale @UIScale: L'instance UIScale à l'intérieur du contenu
  • hoverEffect @string: L'effet de survol à appliquer ("Zoom" ou "Opacity")

Cette fonction permet de lier des effets de survol à un bouton.

Exemple:

local button = LeGUIqueJeVeux:FindFirstChild("MyButton")
local content = button:FindFirstChild("Content")
local scale = content:FindFirstChild("UIScale")

GUI:BindHoverEffect(button, content, scale, "Zoom")

SetButtonVisibility(button: TextButton, visibility: boolean)​

Paramètres:

  • button @TextButton: Le bouton dont on veut définir la visibilité
  • visibility @boolean: L'état de visibilité à définir

Cette fonction permet de définir la visibilité d'un bouton.

Exemple:

local button = LeGUIqueJeVeux:FindFirstChild("MyButton")

GUI:SetButtonVisibility(button, true)  -- Rendre le bouton visible

task.wait(3)
GUI:SetButtonVisibility(button, false)  -- Rendre le bouton invisible

ButtonChangeFrame(button: TextButton, previous: Frame)​

Paramètres:

  • button @TextButton: Le bouton à utiliser pour changer de frame
  • previous @Frame: Le frame précédent

Cette fonction permet de changer de frame en utilisant un bouton spécifique.

Exemple:

local button = LeGUIqueJeVeux:FindFirstChild("MyButton")
local previousFrame = LeGUIqueJeVeux:FindFirstChild("PreviousFrame")

GUI:ButtonChangeFrame(button, previousFrame)
