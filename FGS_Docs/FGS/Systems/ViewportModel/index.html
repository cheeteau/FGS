Le système ViewportModel

Permet de manipuler et d'afficher des modèles dans une vue de type ViewportFrame avec diverses méthodes de calibration et d'ajustement.


Méthodes​

Voici la liste des méthodes dont vous aurez besoin pour vos scripts.

new(vpf: ViewportFrame, camera: Camera)​

Paramètres:

  • vpf @ViewportFrame: La frame de la vue
  • camera @Camera: La caméra utilisée pour la vue

Cette fonction crée une nouvelle instance de ViewportModelClass.

Exemple:

local ViewportModel = self.FGS:GetSystem("ViewportModel")
local GUI = self.FGS:GetSystem("GUI")
local LeGUIqueJeVeux = GUI:GetGUI("NomDeMonScreenGUI")

local vpf = LeGUIqueJeVeux:FindFirstChild("MyButton")
local camera = vpf.CurrentCamera

local modelViewer = ViewportModel.new(vpf, camera)

SetModel(model: Model)​

Paramètres:

  • model @Model: Le modèle à afficher dans la vue

Cette fonction permet de définir le modèle à afficher dans la vue.

Exemple:

local ViewportModel = self.FGS:GetSystem("ViewportModel")
local model = workspace:WaitForChild("MyModel")

modelViewer:SetModel(model)

Calibrate()​

Cette fonction doit être appelée lorsque quelque chose change dans le ViewportFrame ou la caméra, par exemple la taille de la frame ou le champ de vision de la caméra.

Exemple:

local ViewportModel = self.FGS:GetSystem("ViewportModel")

modelViewer:Calibrate()

GetFitDistance(focusPosition: Vector3?)​

Paramètres:

  • focusPosition @Vector3? (Optionnel): La position de mise au point

Cette fonction retourne une distance fixe qui garantit d'encapsuler complètement le modèle. Utile pour tourner librement autour d'un objet sans calculs coûteux.

Exemple:

local ViewportModel = self.FGS:GetSystem("ViewportModel")

local fitDistance = modelViewer:GetFitDistance()
print("Distance d'ajustement:", fitDistance)

GetMinimumFitCFrame(orientation: CFrame)​

Paramètres:

  • orientation @CFrame: L'orientation pour ajuster la vue

Cette fonction retourne le CFrame optimal de la caméra nécessaire pour ajuster au mieux le modèle dans la ViewportFrame à l'orientation donnée.

Exemple:

local ViewportModel = self.FGS:GetSystem("ViewportModel")
local orientation = CFrame.new(0, 0, 0)

local minFitCFrame = modelViewer:GetMinimumFitCFrame(orientation)
print("CFrame d'ajustement minimal:", minFitCFrame)
