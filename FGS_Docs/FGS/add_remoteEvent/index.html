Ajouter un RemoteEvent

Pour rappel, la documentation pour les RemoteEvents est accessible sur ce lien.

Ajouter un RemoteEvent est relativement simple. Il suffit simplement de vous rendre dans le dossier Events présent dans le dossier Remote du sous-dossier Shared du modulescript FGS. Faites attention tout de même de bien nommer le RemoteEvent de manière EXPLICITE afin que tout soit facilement compréhensible pour un autre développeur.
