Ajouter un RemoteFunction

Pour rappel, la documentation pour les RemoteFunctions est accessible sur ce lien.

Comme pour les RemoteEvents, il suffit simplement de vous rendre dans le dossier Functions présent dans le dossier Remote du sous-dossier Shared du modulescript FGS. Faites attention tout de même de bien nommer le RemoteFunction de manière EXPLICITE afin que tout soit facilement compréhensible pour un autre développeur.
